# Ether
An overly minimal Vim-like editor <br/>
Ether uses parts of code from [Kilo](https://github.com/antirez/kilo). Kilo's license is included in this repository.

## How to install?
Run this as root:
```shell
make install
```
Start editing:
```shell
touch file
ether file
```

## How to uninstall?
Run this as root:
```shell
sudo make uninstall
```
